# PROYECTO-FINAL-IA-PARA-LIDERES-B-DAVID-NICOLA
Desarrollo del prototipo del tablero de control en un archivo HTML, con diseño estilo software empresarial moderno, cuatro embarques ficticios de distintos países con sus respectivos estados (dos rojos, uno amarillo, uno verde), panel lateral interactivo con el diagnóstico textual de la IA y botones de acción simulados para cada caso de alerta.
